This application implements the Red/Blue problem.

To run this program on Windows, open 'rbs.sln' and hit F5 to run with debugging.
To modify the commandline arguments, alter them under the Debugging category in Project Properties.


To compile this program on a Unix-style OS, type 'make'.
Then type './linux-rbs' to execute the program (modifying the commandline arguments as necessary).
